## 2. LIVESQL 접속

> - <https://livesql.oracle.com/> 접속
> - 왼쪽 메뉴에서 SQL WORKSHEET 클릭
> - 아래의 코드를 복사하고 붙여넣기

```sql
-- 임플로이(EMPLOYEE) 테이블 생성
-- 사원번호,이름,주민번호,입사일,출신도,전화번호,부서명,직위,급여,보너스
CREATE TABLE emp(
    empNo      VARCHAR2(10) NOT NULL
    ,name      VARCHAR2(30) NOT NULL
    ,rrn       VARCHAR2(14) NOT NULL
    ,hireDate  DATE         NOT NULL
    ,city      VARCHAR2(30) NOT NULL
    ,tel       VARCHAR2(15)
    ,dept      VARCHAR2(30) NOT NULL
    ,pos       VARCHAR2(30) NOT NULL
    ,sal       NUMBER(10)   NOT NULL
    ,bonus     NUMBER(10) NOT NULL
    ,CONSTRAINT pk_emp_empNo PRIMARY KEY(empNo)
    ,CONSTRAINT uk_emp_rrn   UNIQUE(rrn)
);
```

## 3. 실습하기

> - SQL WORKSHEET에 있는 입력 상자를 SQL DEVELOPER라고 생각하고 연습하면 됩니다.

주의사항

1. 날짜 형식 관련된 함수는 동작하지 않을 수 있음. (∵ LIVESQL은 영문 오라클 버전 기준임)
2. 계정 전환 불가 (conn sky/"비밀번호" 등의 계정 전환 불가능합니다)
3. 19c버전으로 알고 있으므로 일부 함수들은 작동하지 않을 수도 있습니다.
4. 브라우저를 껐다 켜거나 오라클 로그아웃을 한 경우 다시 위의 과정부터 진행해야 합니다. (테이블이 초기화됨)
